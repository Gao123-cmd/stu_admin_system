### 3. teach功能
1.完成首页设计，支持显示当前登录用户的姓名。

2.支持教师查看自己的课表，包括课程信息与上课时间地点。

3.完成教师提交成绩的功能，教师可以给选了自己的课程的学生提交成绩。

4.完成教师提交调课申请的功能，教师可以提交调课申请待管理员审核。

5.完成教师审核学生免听申请的功能，教师可以对未审核的学生免听申请进行初步审核，可以进行同意待管理员进一步审核或者直接否决。
